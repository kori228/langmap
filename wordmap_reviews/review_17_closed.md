I am a specialist in Western Austronesian languages (Malayo-Polynesian and Formosan), trained in the comparative-historical methodology established in Blust's *The Austronesian Languages* (revised ed. 2013, with S. ### 3. `su` — fire, hand — Sundanese /eu/ IPA uses /ɤ/ instead of /ɯ/
- **Files:** `words/fire.js` line 205; `words/hand.js` line 199
- **Current:** `su: ["seuneu", "sɤnɤ"]`; `su: ["leungeun", "lɤŋɤn"]`
- **Expected:** `["seuneu", "sɯnɯ"]`; `["leungeun", "lɯŋɯn"]`
- **Why:** In Sundanese, the digraph ⟨eu⟩ represents the high-mid back unrounded vowel /ɯ/ (Nothofer 1975; Walker 1976). The IPA symbol /ɤ/ is the mid back unrounded vowel (as in Bulgarian or Korean), distinct from /ɯ/. This is confirmed internally by the Old Sundanese parallel entries (`osu:` fire line 1032; `osu:` hand line 1026) which correctly use /ɯ/: `["seuneu", "sɯnɯ"]` and `["leungeun", "lɯŋɯn"]`. The main `su:` entries are thus internally inconsistent with their own Old Sundanese counterparts.

### 5. `bbc` — house — *jabu* IPA [dʒabu] is wrong; Toba Batak /j/ = palatal glide
- **File:** `words/house.js` line 872
- **Current:** `bbc: ["jabu", "dʒabu"]`
- **Expected:** `bbc: ["jabu", "ˈjabu"]`
- **Why:** Toba Batak has no native /dʒ/ phoneme. In the Batak script and in all descriptive grammars (Nababan 1981 *A Grammar of Toba-Batak*; Percival 1981), ⟨j⟩ represents the palatal approximant /j/, not an affricate. The form *jabu* (house, cognate with PAN \**Rumaq* via metathesis?) is [ˈjabu], not [ˈdʒabu]. The transcription appears to have been generated by analogy with Javanese/Indonesian ⟨j⟩ = /dʒ/, which does not apply to Toba Batak.

### 7. `tl` (Tagalog) — systematic omission of final glottal stop across multiple entries
- **Files:** `words/heart.js` line 119; `words/dog.js` line 119; `words/cat.js` line 119; `words/good.js` line 119; `words/one.js` line 119
- **Current examples:** `tl: ["puso", "puso"]`; `tl: ["aso", "aso"]`; `tl: ["pusa", "pusa"]`; `tl: ["mabuti", "mabuti"]`; `tl: ["isa", "isa"]`
- **Expected:** `["puso", "puˈsoʔ"]`; `["aso", "ˈʔasoʔ"]`; `["pusa", "puˈsaʔ"]`; `["mabuti", "maˈbutiʔ"]`; `["isa", "ˈʔisaʔ"]`
- **Why:** In Tagalog, the final glottal stop is phonemic (Schachter & Otanes 1972 §1.4). All of *puso*, *aso*, *pusa*, *mabuti*, and *isa* carry a final /-ʔ/ that conditions morphophonological alternations (e.g., *puso* vs. *pusô* in older orthography, and *puso + -ng* → *pusong* with deletion rules). Kapampangan parallel entries in the same data set correctly mark this glottal (cf. `pam: ["pusu", "ˈpusoʔ"]`), making the Tagalog omission an inconsistency.

### 8. `ceb` — good — *maayo* IPA [maːjo] omits medial glottal stop

Treating the sequence as a simple long /aː/ misrepresents the phonology. This is internally inconsistent: Hiligaynon *maayo* in the same dataset is correctly transcribed [maʔajɔ] (line 577), confirming the expected form.

### 13. `min` — one — *ciek* IPA [tʃie] truncates the final stop

- **Current:** `min: ["ciek", "tʃie"]`
- **Expected:** `min: ["ciek", "tʃieʔ"]`

This is a regular phonological rule shared with many Western Malay dialects. The form *ciek* should therefore be transcribed [tʃieʔ]. ### 14. `jv` — register inconsistency: *matur nuwun* (krama) vs ngoko for all other entries
- **Files:** `words/thanks.js` line 197 vs `words/dog.js` line 197; `words/fire.js` line 203; `words/house.js` line 204; etc.
- **Observation:** All Javanese entries consistently use ngoko (informal register): *asu* "dog", *geni* "fire", *omah* "house", *mangan* "eat", *ngombe* "drink". However, the Javanese "thanks" entry uses *matur nuwun*, which is krama (polite register) — the krama equivalent of ngoko *maturnuwun* / *suwun*. The conventional ngoko thanks is *suwun* [ˈsuwun].
- **Why this matters:** Mixing registers across a lexical set misrepresents the internal structure of Javanese speech-level grammar (Soepomo 1968; Robson 2002 *Javanese Grammar for Students*). Either all entries should be ngoko, or all should be krama (with a note). The current dataset uses ngoko everywhere except thanks.

### 17. `bbc` — good — IPA [deŋːan] misrepresents /ng/ + /g/ as a long nasal
- **File:** `words/good.js` line 865
- **Current:** `bbc: ["denggan", "deŋːan"]`
- **Expected:** `bbc: ["denggan", "deŋgan"]`
- **Why:** Toba Batak *denggan* "good, fine" consists of the sequence /ng/ + /g/ (velar nasal + velar stop), not a single long/geminate nasal. The IPA transcription [deŋːan] implies a lengthened nasal with no following stop, which is phonologically inaccurate. The correct representation is [deŋgan] or [deŋ.gan], reflecting the heterosyllabic nasal+stop cluster. Compare English "finger" [fɪŋɡər] where the nasal is not lengthened. Geminate notation [ŋː] is reserved for long nasals without a stop onset on the following syllable.

### 18. `ilo` — thanks — *agyamanak* is a first-person-singular inflected form
- **File:** `words/thanks.js` line 201

Presenting a 1SG inflected form as the citation is inconsistent with how other Philippine entries handle the same concept.
