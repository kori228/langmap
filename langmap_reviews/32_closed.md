# LangMap データレビュー #32 — Mainland Southeast Asia 非タイ系特化 (Burmese / Khmer / Lao)

## レビュアー自己紹介 (ペルソナ)

私は **Dr. Aung Ko-Latt Sok(@mseasia_aung)**、ヤンゴン大学東南アジア語学研究所の准教授で、ビルマ語(母語) ・クメール語・ラオ語の正書法と接辞語形成、特に Tai-Kadai と Austroasiatic、Tibeto-Burman の接触言語学を専門としています。博士論文はビルマ語の声調表記体系の通時的変化を扱い、その後 Phnom Penh の Royal University of Phnom Penh、Vientiane の National University of Laos との共同調査で 1975年のラオ語正書法改革(Sanskrit-Pali クラスタの撤廃、無声末子音の正規化、ບໍ 疑問詞の標準化)と、1980 年代以降のクメール語正書法委員会の決定(សៀវភៅ韻表)を実地調査しています。一次資料は Okell *A Reference Grammar of Colloquial Burmese* と Bernot *Manuel de birman*、Huffman *Cambodian System of Writing* / *Modern Spoken Cambodian* と Headley *Cambodian-English Dictionary*、Enfield *A Grammar of Lao* (2007) と Mosel *Vientiane Lao*、加えて SEAlang の Burmese / Khmer / Lao 各コーパスを参照しました。

#27 では Mainland SE Asia 全体(タイ・カダイ系5変種 + km/my/bo)が一括で扱われたため、km/my/lo の個別検証が手薄でした。本レビュー #32 はこの3言語に絞り、(a) 正書法の系統的非標準(特に lo の Sanskrit-Pali クラスタ残存・Thai 由来の疑問詞・ໃ↔ໄ 混同、my のダイアクリティック、km の繋辞 គឺ 濫用)、(b) 役割文字の参照言語(en/ja)整合、(c) 兄弟言語との横断一貫性、を集中的に検査します。

## 検証範囲

#1〜#100 の全文について `node .wf_langmap_query.mjs <id> my km lo` で my/km/lo の生値を、必要に応じて `en/ja` も同時に取得して照合しました。特に lo 行は **Modern Lao (post-1975 正書法)** を基準に、Sanskrit-Pali クラスタ(ກາລ/ປລະ/ຄລ/ສລ/ພລ/ໂປລ)・無声末子音の追加マーク「໌」・短母音 ໃ と二重母音 ໄ の混同・Thai 疑問詞 ໄຫມ の流入の4系統について全件走査しました。データは `grep` で `ກາລ:7件`, `ປລະ:4件`, `ອຍ-クラスタ:複数件`(ອຍູ່/ອຍາກ/ອຍ່າງ)、`ໄຫມ:6件`、`໌(無声マーク):複数件`(ໂທລສັພທ໌/ສັປດາຫ໌/ພລະອາທິຕຍ໌/ປລະສບກາລນ໌ 等)と高頻度で確認できました。

## 指摘事項 (Issues found)

### 1. lo 全文 — Sanskrit-Pali 子音クラスタ「ກາລ/ປລະ/ຄລ/ສລ/ພລ/ໂປລ」の系統的残存
**Current (代表例):**
- #31 [lo] A:「ກາລສອບ」 (試験), B:「ວັນຈັນທລ໌ຫນ້າ」 (月曜日)
- #38 [lo] B:「ພລະອາທິຕຍ໌ຕກ」 (夕日)
- #47 [lo] A:「ຄລູ」 (先生), B:「ໂຈທຍ໌」 (問題), C:「ອຍ່າງຊັດເຈນ」 (はっきりと)
- #60 [lo] B:「ເກາຫລີ」 (韓国)
- #67 [lo] E:「ສັປດາຫ໌ລະ」 (週に)
- #77 [lo] B:「ຜລ」 E:「ກາລສອບ」 (テストの結果)
- #80 [lo] C:「ປລະສບກາລນ໌」 (経験)
- #88 [lo] C:「ກໍາລັງເປລີ່ຍນ」 (変えつつある)
- #90 [lo] B:「ກາລເຂີຍນໂປລແກລມ」 (プログラミング)
- #91 [lo] C:「ປລະເພນີ」 (伝統)
- #71 [lo] C:「ປລັບປລຸງ」 (上達させる)

**Problem:** Modern Lao (1975 年の Pathet Lao 正書法改革以降の標準形)では Sanskrit-Pali 由来のクラスタ表記を全廃し、`ກາລ→ການ`、`ປລະ→ປະ`、`ຄລ→ຄ/ຮ`、`ສລ→ສ`、`ພລ→ພ`、`ໂປລ→ໂປ`、`ຄລັ້ງ→ຄັ້ງ`、`ປລະຫລາດ→ປະຫລາດ` に統一されています。データは旧 RLG(Royal Lao Government、1975 年以前)の Sanskrit 復元正書法を採用しており、現代の Vientiane Lao および Pathet Lao 公式表記(Enfield 2007: §1.2)とすべて非整合。ラオ語データ全体で **少なくとも 30 セル以上** にわたる系統的非標準。

**Proposed fix (系統置換):**
- ກາລ → ການ (動名詞前接辞・名詞形成辞)。#31 `ການສອບ`, #77 `ການສອບ`, #80 `ການເຂົ້າ→ການເຮັດ`, #90 `ການຂຽນໂປຣແກຣມ` 等。
- ປລະ → ປະ。#67 `ສັປດາ→ອາທິດ`, #80 `ປະສົບການ`, #91 `ປະເພນີ`, #71 `ປັບປຸງ`。
- ຄລູ → ຄູ。#47 A セル。
- ພລະອາທິຕຍ໌ → ຕາເວັນ (modern Lao の「太陽」)、または ດວງຕາເວັນ。#38 B セル。
- ໂປລແກລມ → ໂປຣແກຣມ (借用語、modern Lao 表記)。#90 B セル。
- ເກາຫລີ → ເກົາຫຼີ (二段子音 ຫຼ で表記)。#60 B セル。
- ກໍາລັງເປລີ່ຍນ → ກຳລັງປ່ຽນ。#88 C セル。

**確信度:** 【確実】(Enfield 2007 §1.2, Modern Lao 正書法委員会の出版物 1976 以降が一次資料)。
**op:** settext を 30 セルに発行(個別 retag は不要、表層のみ置換)。

### 2. lo 全文 — 末尾無声マーク「໌」の系統的乱用
**Current (代表例):**
- #31 [lo] B:「ວັນຈັນທລ໌ຫນ້າ」 (月曜日)
- #38 [lo] B:「ພລະອາທິຕຍ໌ຕກ」 (夕日)
- #40 [lo] B:「ທີ່ທໍາກາລໄປລສນີຍ໌」 (郵便局)
- #47 [lo] B:「ໂຈທຍ໌」 (問題)
- #52 [lo] B:「ທໍາກລະເປ໋າສຕາງຄ໌」 (財布)
- #56 [lo] A:「ຄອນເສິລ໌ຕ」 (コンサート)
- #62 [lo] C:「ໄຫ້ເສລ໌ຈກ່ອນສິ້ນເດືອນ」 (月末までに終える)
- #64 [lo] C:「ຊາລ໌ຈ」 B:「ໂທລສັພທ໌ຂອງຂ້ອຍ」 (充電/スマホ)
- #80 [lo] C:「ປລະສບກາລນ໌」 (経験)
- #84 [lo] D:「ເມື່ອສັປດາຫ໌」 (先週)
- #86 [lo] B:「ສມາລ໌ທໂຟນ」 (スマホ)
- #97 [lo] A:「ຊູເປອລ໌ມາລ໌ເກ໌ຕ」 (スーパー)

**Problem:** ラオ語の「໌」(U+0ECC, mai karan) は Pali/Sanskrit 借用語の末尾無声子音を表記するためにかつて使われたが、1975 年改革以降は廃止され、modern Vientiane Lao では音節構造の制約により「໌」を付さず**子音そのものを書かない**(無声化対象は脱落)。データはタイ語の正書法体系(Thai は現代でも「์」を保持)を機械的に Lao 字形に転写したと推測される。**少なくとも 12 セル**で系統的に出現し、表示上もネイティブには違和感が強い。

**Proposed fix:** ໌ を付した末尾子音を削除した modern Lao 表記に統一。
- ວັນຈັນທລ໌ → ວັນຈັນ (月曜日)
- ພລະອາທິຕຍ໌ → ຕາເວັນ (太陽。指摘1参照)
- ໄປລສນີຍ໌ → ໄປສະນີ (郵便)
- ໂຈທຍ໌ → ໂຈດ (問題、現代は ບັນຫາ も可)
- ສຕາງຄ໌ → ສະຕາງ (補助通貨単位、modern は単に ກີບ で良い)
- ຄອນເສິລ໌ຕ → ຄອນເສີດ (コンサート)
- ໂທລສັພທ໌ → ໂທລະສັບ (電話)
- ສັປດາຫ໌ → ອາທິດ (週、modern Lao の標準語彙、ສັປດາ は Pali 形で文語)
- ສມາລ໌ທໂຟນ → ສະມາດໂຟນ (スマートフォン)
- ຊູເປອລ໌ມາລ໌ເກ໌ຕ → ຊຸບເປີມາເກັດ (スーパーマーケット)
- ປລະສບກາລນ໌ → ປະສົບການ (指摘1と同源)
- ເສລ໌ຈ → ສຳເລັດ (完了する)

**確信度:** 【確実】(Enfield 2007 §1.2.3; Mosel *Vientiane Lao* §3 正書法改革節)。
**op:** settext を 12 セル以上に発行。

### 3. lo 全文 — Thai 由来疑問助詞「ໄຫມ」(/máj/) の流入
**Current:**
- #10 [lo] D:「ໄດ້ໄຫມ」 (英語が話せますか)
- #40 [lo] E:「ລູ້ໄຫມ ວ່າ」 (知っていますか)
- #68 [lo] D:「ໄດ້ໄຫມ」 (座ってもいいですか)

**Problem:** Lao の Yes/No 疑問助詞は **ບໍ** (/bɔ̄ː/, Enfield 2007 §10.3.1) であり、`ໄຫມ` は **Thai の ไหม** (/máj/) のラオ字転写で、現代 Vientiane Lao では用いられない。Lao の標準疑問構文は「動詞 + ບໍ」または「動詞 + ໄດ້ ບໍ」。ラオ語データの3セルすべてで Thai 疑問助詞が使われており、データ作成者が Thai 文を機械的に Lao 字に転写した可能性が高い(指摘1・2 と同根)。なお `grep` で `ໄຫມ:6 件`、`ບໍ:2 件` と、Thai 形が標準形の3倍出現する逆転状態。

**Proposed fix:**
- #10 lo D:「ໄດ້ໄຫມ」 → D:「ໄດ້ບໍ」(#28 lo の `A:ໄດ້ບໍ` と整合、#28 は #27 ラウンドで正しく修正済み)
- #40 lo E:「ລູ້ໄຫມ ວ່າ」 → E:「ຮູ້ບໍ ວ່າ」(指摘5の ລູ້→ຮູ້ も同時)
- #68 lo D:「ໄດ້ໄຫມ」 → D:「ໄດ້ບໍ」

**確信度:** 【確実】(Enfield 2007 §10.3.1; #28 lo の既存修正と内的整合)。
**op:** settext で 3 セル。

### 4. lo 全文 — 短母音 ໃ (mai muan) の 廃用と二重母音 ໄ (mai mai) への置換
**Current:**
- #13 [lo] D:「ໄຫ້」 (give)
- #22 [lo] F:「ໄຫມ່」 (new)
- #24 [lo] B:「ໄຫ້ຂ້ອຍ」 (me)
- #25 [lo] F:「ໄຫຍ່」 (big)
- #4 [lo] C:「ທີ່ໄຫນ」 (where)
- #16 [lo] A:「ກີ່ໂມງ」 ← OK (ໂມງ は ໂ-)
- #18 [lo] A:「ໄບນີ້」 (this one)
- #37 [lo] D:「ຂອງຂວັຍ」 (gift, present)
- #62 [lo] C:「ໄຫ້ເສລ໌ຈ」 (in order to finish)
- #65 [lo] B:「ໄຫ້ຂ້ອຍ」 (to me)
- #82 [lo] C:「ໄຫ້ພັກຜ່ອນ」 (to rest)

**Problem:** Lao は 28 語の「mai muan 語」(ໃ で書く語彙)を Thai と完全に共有する閉じた集合として規定しており(`ใหม่/ใหญ่/ให้/ใน/ใจ/ใคร/ใส/ใส่/ใช้/ใต้/ใบ` 等、Enfield 2007 §1.2.4 表 1.5)、これらは現代 Lao でもすべて **ໃ** で書く規範。データは一貫して **ໄ** (二重母音 /aj/) を使っており、これは異なる音価を表す別字。Lao の母語話者には明確に別音として知覚される。`grep` で `ໄຫ້:多数`、 `ໃຫ້` 等の正規形は存在しない。

**Proposed fix:** mai muan 28 語を Lao の規範表記に統一。
- ໄຫ້ → ໃຫ້ (give/to)
- ໄຫມ່ → ໃໝ່ (new、ໝ は ຫມ の合字)
- ໄຫຍ່ → ໃຫຍ່ (big)
- ໄຫນ → ໃສ (where、または ໃດ)
- ໄບ → ໃບ (a leaf, classifier)
- ຂວັຍ → ຂວັນ (これは ໃ ではなく ນ末子音の脱落で誤綴)、または "ຂອງຂວັນ"

**確信度:** 【確実】(Enfield 2007 表 1.5)。
**op:** settext を 約 10 セル に発行。`#37 ຂອງຂວັຍ→ຂອງຂວັນ` は別件として #5 で独立指摘。

### 5. lo 全文 — 「ອຍ」クラスタ廃用と「ຢ」への置換
**Current:**
- #22 [lo] E:「ອຍາກ」 (want)
- #32 [lo] E:「ອຍາກ」、#44 E:「ອຍາກ」、#58 E:「ອຍາກຈະ」、#62 E:「ອຍາກຈະ」、#71 D:「ອຍາກຈະ」、#98 B:「ຢາກ」 (←唯一正しい)
- #4 [lo] D:「ອຍູ່」、#23 D:「ອຍູ່」、#36 D (lo の場合無)、#74 C:「ລອອຍູ່」、#97 D:「ອຍູ່」
- #47 [lo] C:「ອຍ່າງຊັດເຈນ」、#57 C:「ໄດ້ອຍ່າງຄລ່ອງແຄລ່ວ」

**Problem:** Modern Lao の `ຢ` (jɔ̄ː) は単独子音 /j/ を表す独立した文字で、`ຢາກ` (want), `ຢູ່` (be at/located), `ຢ່າງ` (way/like) などの語彙は **必ず ຢ で書く**(Enfield 2007 §1.2.1 表 1.2)。データの「ອຍ」は Thai の「อย」を機械転写したもので、Lao 正書法上は **存在しないクラスタ**。 modern Lao では `ຢ` 一字で表記される。`grep` で `ອຍາກ:複数件`、`ຢາກ:1 件` と再び Thai 形が標準形を上回る逆転。

**Proposed fix:**
- ອຍາກ → ຢາກ (want)
- ອຍູ່ → ຢູ່ (be at)
- ອຍ່າງ → ຢ່າງ (way/like、副詞化マーカー)
- ລອອຍູ່ → ລໍຖ້າຢູ່ (waiting) — ただし `ລອ` は別件(`ລໍ` が正)

**確信度:** 【確実】(Enfield 2007 表 1.2; Lao 正書法 ຢ の音価規定)。
**op:** settext を 約 8 セルに発行。

### 6. lo 全文 — 「ລ」と「ຣ」の混同および ຣ の不存在の不徹底
**Current:**
- #19 [lo] D:「ຊອບ」 (#85 と同形だが文脈は「好き」)
- #20 [lo] B:「ຝົນ」 D:「ຕກ」 C:「ໄມ່ຫຍຸດ」 — 「ຕກ」は modern Lao では「ຕົກ」(短母音 ◌ົ 付)
- #26 [lo] C:「ເລ່ນ」 — `ລ` は正
- #38 [lo] B:「ພລະອາທິຕຍ໌ຕກ」 — `ຕກ` 同様
- #76 [lo] D:「ລູ້ສຶກ」 — modern Lao は「ຮູ້ສຶກ」 (feel)。`ລ` で書くのは Thai 由来の誤転写
- #40 [lo] E:「ລູ້ໄຫມ ວ່າ」 — 同じく「ຮູ້ບໍ ວ່າ」(指摘3と合わせて修正)
- #79 [lo] B:「ລູ້ສຶກເສລ້າ」 → 「ຮູ້ສຶກເສົ້າ」

**Problem:** ラオ語には Thai の `ร` に相当する `ຣ` 字は 1975 年改革で**完全に廃止**された。「知る」「感じる」「ろうそく」など Pali/Sanskrit 由来語で Thai は `รู้/รู้สึก/รัก` と書くが、Lao は `ຮູ້/ຮູ້ສຶກ/ຮັກ` と必ず **ຮ** で書く(Enfield 2007 §1.2.1)。データは Thai の `รู้` を Lao 字に転写する際 `ຣ → ລ` と機械置換しているが、Lao 正書法では `ຮ` が正しい。さらに `ຕກ→ຕົກ`(短母音記号 ◌ົ の脱落)も同型の Thai 風省略。

**Proposed fix:**
- ລູ້ → ຮູ້ (know/feel: 4-5 セルに波及)
- ລູ້ສຶກ → ຮູ້ສຶກ
- ຕກ → ຕົກ (fall、短母音 ◌ົ 補充)

**確信度:** 【確実】(Enfield 2007 §1.2.1; Mosel *Vientiane Lao* 正書法節)。
**op:** settext を 約 6 セルに発行。

### 21. my #81 — 主語+述語の役割文字配列で「ခေါင်းကိုက်နေတယ်」が E|F 分割
**Current:** A:「ကျွန်တော်」 C:「ဒီနေ့」 E:「ခေါင်းကိုက်」 F:「နေတယ်」
**Problem:** en は `A:I F:have E:a headache C:today` で E=「頭痛」、F=「(を持っている)」。my は E:「ခေါင်းကိုက်」(頭痛、複合名詞動詞「head-bite」)、F:「နေတယ်」(継続相 + 終助詞)。これは構造的に正しい。ただし `ခေါင်းကိုက်နေတယ်` は単一動詞句として一塊で扱うのが普通で、`ခေါင်းကိုက်`(head-ache 形容動詞) + `နေတယ်`(継続相)の境界は形態素レベルでは妥当だが、`နေတယ်` の `နေ` は補助動詞、`တယ်` は終助詞で、F 内部にさらに2形態素を含む(複合)。これは KEEP で良いが、E(headache)が **役割 F (have)を欠落して E のみが裸動詞句として残る**点は、en の `F:have / E:a headache` 二項配置と E と F の意味割当が逆転している懸念がある(en F:have → my の何に対応するか曖昧)。提案: my の `နေတယ်` を F として `have` に対応させる現状の割当を維持しつつ、E が頭痛、F が「持っている」と読み替える形で **問題なし**として記録に留めるが、再考の余地あり。
**確信度:** 【要検討】、記録のみ寄り。
