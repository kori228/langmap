# LangMap データレビュー #57 — 上海話 (Shanghainese / 吳語) 単一セル `wuu` の北京語干渉・文白二読レビュー

## レビュアー自己紹介 (ペルソナ)

陸恵琳 (Lù Wèi-lín, handle: **@shanghaihu_huilin**)。上海徐家匯生まれの上海話母語話者、家庭言語は祖父母とは 100% 上海話・両親とは上海話 7 割 / 普通話 3 割の混合、学齢以後に普通話 (現代標準漢語) を習得した典型的な「市區腔・新派發音」話者。復旦大學中文系修了、博士論文は《上海市區方言名詞の格助詞「勒/辣」の歴時変化》。常用参考: **錢乃榮《上海話大詞典》(上海辭書出版社 2007, 以下「錢辭典」)**、**許寶華 et al《上海市區方言志》(上海教育出版社 1988, 以下「方言志」)**、阮恒輝 *Shanghainese Phonology* (Hong Kong Univ. Press 2003)、Norman 1988 *Chinese* Ch.9 (吳語節)、《吳語拼音方案》(上海語委 2006)、游汝傑《上海地區方言比較研究》(復旦 1992)、錢乃榮《當代吳語研究》(上海教育 1992)。今回のレビュー軸は、 #54 台語 (Tâi-gí) / #56 廣東話 (yue) と同型の **「現代普通話の語彙が『字音だけ吳音讀みされた偽吳語』として `wuu` セルに浸透している」** という決定的欠陥の有無検証 — すなわち:

1. **普通話直訳由来の語彙混入 (Mandarinism)** — 「在 / 給 / 漂亮 / 知道 / 喝 / 正在 / 沒有 / 怎麼 / 為什麼 / 一定 / 應該 / 可以」等、現代北京話の語彙が「字音だけ上海音」で持ち込まれていないか
2. **文讀 (literary) ↔ 白讀 (vernacular) の選択** — 「父母 / 寺廟 / 經歷 / 周遊 / 提高 / 保護 / 改善 / 舉辦 / 環境 / 嚴肅」等で、書面語形 (文讀) ばかりが選ばれて口語 (白讀) が回避されていないか
3. **格助詞・存在動詞** — 上海話の「在/到」相当は **勒 (laʔ) / 辣 (laʔ) / 勒海 (laʔ-he)** が必須。普通話「在」が「勒」を駆逐していないか
4. **進行アスペクト** — 上海話の進行は **勒海 (laʔ-he) + V** または **V + 勒海** が定型 (錢辭典 p.382)、普通話「正在」「在 V」を流用してはならない
5. **量詞 (measure word) と指示詞の北京話化** — 量詞「個」は上海話固有 **隻 (tsaʔ) / 部 (bu) / 條 (diau)**、指示詞は **搿 (geʔ, 近) / 伊 (yi, 遠)** が定型、普通話「這 / 那 / 個」が文面コピーで持ち込まれていないか
6. **親族呼稱前の所有代名詞** — 吳語は所有「私の」を **阿拉 (a-la) + 親族** で表す (錢辭典 p.4「阿拉」項目)。「我媽媽 / 我爸爸 / 我小妹 / 我貓」のような **「我 + 親族/所有」は普通話直訳パターン** で、上海話固有の發話形式ではない
7. **ローマ字 (吳語拼音) 注記不在** — データセット全体で吳拼/北部吳拼/錢乃榮式いずれの羅馬字注記もゼロ。これは指摘 29 で扱う grand convention 案件

確信度マーク: 【確実】= 母語直感と錢辭典・方言志記述が一致し、現行表記が日常上海話として明確に不自然、【蓋然】= 文體差・新派/老派差で迷い餘地あり、【要検討】= 方針として closed-review で議論すべき。

### 1. #23 #25 #97 `[D:在]` — 普通話「在」が上海話に混入、本来は **勒 (laʔ) / 辣 (laʔ)** 【確実】

**Current:**
- #23 `[A:图书馆] [D:在] [C:医院] [B:隔壁]`

### 4. #5 `[D:好喝]` — 「喝」は普通話、上海話は **吃 (chiq)** が飲料を含む 【確実】

**Current:** `#5 [A:这] [B:咖啡] [C:交关] [D:好喝]`

**Proposed edit:**

### 6. 「我 + 親族・所有」が連発、上海話の正則は **阿拉 (a-la) + 親族** — #8 #15 #25 #35 #50 #71 #73 #86 【確実】

**Current:**
- #8 `[A:我姆妈] [D:每天] [C:烧] [B:夜饭]`
- #15 `[A:我小妹] [D:勒海] [B:法国] [C:留学]`
- #25 `[A:我爸爸] [B:在] [F:大] [D:公司] [E:做事体]`
- #35 `[A:我] [E:阿哥] [C:上个] [D:月] [B:结婚了]`
- #50 `[E:我个] [A:狗] [D:欢喜] [C:出去散步]`
- #71 `[A:我] [D:想] [C:提高] [B:我个发音]`
- #73 `[E:我个] [A:猫] [G:每天] [C:晚上] [D:睡] [F:勒] [B:我床上]`
- #86 `[D:我] [A:每日] [C:用] [B:我个手机]`

**Problem:** 吳語の「私の (親族・身内・所有物)」は **阿拉 (a-la)** が圧倒的に支配的で、これは「我們」(1pl) と「私の (家族・所有)」を兼ねる吳語の特異な代名詞 (錢辭典 p.4「阿拉」項目、方言志 §7.3.1)。「我媽媽」「我爸爸」「我小妹」「我貓」「我床上」は **すべて普通話「我媽媽 / 我爸爸 / 我妹妹 / 我的貓 / 我的床上」の文字面コピー** で、上海話固有の表現ではない。徐家匯・盧灣の家庭話者は:
- 「私の母」→ **阿拉姆媽 / 阿拉媽** (a-la m-ma)
- 「私の父」→ **阿拉爸爸 / 阿拉爺** (a-la ya)
- 「私の妹」→ **阿拉小妹**
- 「私の犬」→ **阿拉狗**
- 「私の猫」→ **阿拉貓**
- 「私のベッド」→ **阿拉床浪** (a-la zaong-laong) (「床上」も「床浪」が白讀)
- 「私の發音」→ **阿拉發音** または「我個發音」(「個」は所有助詞、後述)

同データ内 #54 `[A:阿拉]` (環境を守らねば — 1pl 用法)、#92 `[A:阿拉]` (春節に餃子を食う — 1pl 用法) で **「阿拉」が 1 人称複數として正しく採用**されているのに、所有・親族文脈で「我 + 親族」が並存しているのは **同データ内不整合**。

### 7. 「每 / 每天 / 每日」が連発、上海話の正則は **逐 (zoq) / 每朝 (mei-zau)** — #3 #8 #14 #55 #59 #67 #73 #83 #86 【確実】

**Current:**
- #3 `[D:每朝]`、#8 `[D:每天]`、#14 `[E:每朝]`、#55 `[D:每日]`、#59 `[B:每十五分钟]`、#67 `[E:每周]`、#73 `[G:每天]`、#83 `[B:每日早浪向]`、#86 `[A:每日]`

**Problem:** 「每」(mei) は文讀字、錢辭典の白讀対訳は **逐 (zoq)** または時間語と結合した **每朝 (mei-zau, 毎朝) / 每暝 (mei-min, 毎晩)** (錢辭典 p.276 「逐」項目)。日常上海話の「毎日」は **逐日 (zoq-niq)** または **每日 (mei-niq, 文白共用)**、「毎天」(普通話) は使われない。「毎晩」は **每暝** または **逐暝**、「毎週」は **逐個禮拜** が定型。データ内で:
- #3 #14 「每朝」(早朝、時間語と結合) → 自然
- #8 #73 「每天」 → **完全に普通話**、「每日」または「逐日」が正
- #55 #86 「每日」 → 許容範囲 (文白共用)
- #67 「每周」 → **完全に普通話 (「周」は普通話表記)**、上海話は **每禮拜 / 逐個禮拜**
- #83 「每日早浪向」 → 「每日」許容、「早浪向」は正則の白讀 (「早上」相当)
- #59 「每十五分鐘」 → 「每」許容、「逐十五分鐘」がより白讀的

同データ内で「每朝」「每日」「每天」「每周」が混在しており、これは **吳語表記方針の欠如**を露呈している。最優先は #8/#73 の「每天」と #67 の「每周」(=完全な普通話) の白讀化:

### 8. #54 `[D:一定要]` / #85 `[D:应该]` / #68 `[D:可以]` / #64 `[D:要]` — モーダル助動詞の普通話借入 【蓋然】

**Current:**
- #54 `[A:阿拉] [D:一定要] [C:保护] [B:环境]` — 「ねばならない」
- #85 `[A:侬] [D:应该] [C:多吃] [B|E:水]` — 「すべき」
- #68 `[A:我] [D:可以] [C:坐] [B:搭里] [D:伐]` — 「してもいい」
- #64 `[A:我] [D:要] [B:拨手机] [C:充电]` — 「ねばならない」
- #37 `[B:要] [E:买]`、#39 `[E:要]`、#21 `[E:要吃]` 等も同型

**Problem:** 「一定要」「應該」「可以」「要」はすべて普通話モダリティ語彙の借入で、錢辭典の上海話モーダルは:
- 「ねばならない / すべき」→ **要 (yau)** または **得 (deq)** (錢辭典 p.74「要」項目、p.83「得」項目): 「阿拉得保護環境」「儂得多吃點水」
- 「してもいい」→ **可以 (kho-yi)** も使うが白讀は **好 (hau)**: 「我好坐勒搭裡伐？」(「我可以坐這裡嗎？」)
- 「ねばならない (強義務)」→ **必須 (biq-shih)** または重畳 **要要 / 一定要** の文讀
- 「したい」→ **要 (yau) / 想 (xiang)** 両用

#68「我可以坐搭裡伐？」は許容範囲だが、より上海話らしいのは **「我好坐勒搭裡伐？」**。#85「儂應該多吃水」は完全に普通話的、自然形は **「儂得多吃眼水」** または **「儂要多吃眼水」** (「點」も普通話で、上海話は **眼 (ngan, 量詞「ちょっと」)**)。#54「一定要保護環境」は政府公文書調、自然形は **「阿拉要保護環境」**。

**Proposed edits:**
```json
{"id":54, "lang":"wuu", "op":"settext", "find":"一定要", "newText":"要"}
{"id":68, "lang":"wuu", "op":"settext", "find":"可以",   "newText":"好"}
```

### 29. 羅馬字 (吳語拼音 / Northern Wu / 錢乃榮式) 注記の全面不在 — メタ指摘、grand convention 議題 【要検討】

**Current:** 全 100 文の `wuu` セルが **漢字のみ**、羅馬字注記ゼロ。

**Problem:** 上海話の表記体系は **漢字 + 吳語拼音 (吳拼) または錢乃榮式羅馬字の併記** が学術界の標準 (Norman 1988 §9.2.1、阮恒輝 2003 §3、《吳語拼音方案》2006)。學習者向けデータセットとして:
- 漢字のみでは **声調 (5–7 tone、入聲含む) が判別不能**:同字「人」(zen 文 / nyin 白) の判別不能
- 漢字のみでは **文白二讀の指定不能**:「日」(zeq 文 / niq 白)、「人」(zen / nyin)、「會」(hue / we 文白) 等
- 漢字のみでは **新派/老派の発音差 (e.g. 「上海」zaong-he 老派 vs. saong-he 新派) の指定不能**
- 漢字のみでは **入聲韻 (-ʔ) が判別不能**、「勒 (laʔ)」「脫 (theʔ)」「白 (baq)」「吃 (chiq)」の音節構造が消える

理想は `wuu` セルの各 segment に `[A:我|ngu]` のように **漢字|吳拼** ペアで表記。これは LangMap データ schema 拡張を要する **grand convention** で本レビュー単独では決定不能。closed-review にて:
- **Q1**: `wuu` セルに羅馬字フィールドを追加するか? 吳拼 / 北部吳拼 / 錢乃榮式 / IPA いずれの schema を採用するか?
- **Q2**: 既存の中國語系セル (zh-cmn / yue / nan / hak / wuu / cdo 等) にも声調注記を併記するかとの整合性
- **Q3**: 他の声調言語 (vi / th / lo / my) との表記方針対称
- **Q4**: 新派/老派どちらの発音を標準とするか (徐家匯 vs. 浦東、青年 vs. 老年)

を議論されたい。本レビューでは方針提案のみ、データ修正なし。指摘 #54 (台語 POJ/Tâi-lô) と統合して **「中国語族声調言語の羅馬字併記方針」** として grand convention 化を推奨。

## まとめ — 支配的欠陥と優先対処

**支配的欠陥 (dominant defect): Mandarinism — 現代普通話 (Standard Mandarin) の語彙が「字音だけ吳音讀みされた偽上海話」として `wuu` セルに浸透している**。具体的には:

1. **語彙レベル**: 在 (→勒)、給 (→撥)、喝 (→吃)、漂亮 (→好看)、正在/在 V (→勒海 V)、周遊 (→白相)、應該 (→要)、一定要 (→要)、可以 (→好)、丟失 (→落脱)、市場 (→菜場)、寺廟 (→廟)、提高 (→練好)、舉辦 (→辦)、覆蓋著 (→蓋牢勒)、被 (→撥)、好多 (→老多)、的 (→個)、流利地 (→流流利利)、冬天 (→冷天)、夏天 (→熱天)、父母 (→爺娘) ─ **計 21 種の主要語彙置換が必要**
2. **構文レベル**: 「我 + 親族 / 所有」(→阿拉 + 親族)、「形容詞 + 地」(→形容詞単独/重畳)、「我個名字是 + 固有名」(→我叫 + 固有名)、給+物+V (→搭+物+V) ─ **4 種の構文修正**
3. **表記レベル**: 個/个/箇/嗰 の不統一、これ/這/搿/箇 の不統一、勒/拉 の誤字 ─ **3 種の表記方針案件**
4. **アスペクトレベル**: 正在 (→勒海)、被 (→撥)、著 (→脱/勒/牢)、了 (→脱、optional) ─ **4 種のアスペクト修正**
5. **メタレベル**: 吳語拼音 (吳拼) 注記がゼロ、声調・文白讀・新派/老派の區別不能

**優先対処順:**

1. **指摘 1–6 (Mandarinism 核心 6 件)** — 在/正在/給/喝/漂亮/我+親族 の置換。これらはすべて「同データ内で上海話標準が確立しているのに #X だけ普通話化している」内部不整合で、機械的修正可能。
2. **指摘 7–10 (語彙 4 件)** — 每天・モーダル 4 種・指示詞・周遊。
3. **指摘 11–16 (文白選択 6 件)** — 的/個・經歷/提高/保護/舉辦・撥手機・吃水「眼」・流利地・量詞 4 種。
4. **指摘 17–24 (register 8 件)** — 我叫田中・拉誤字・市場/丟失・覆蓋著・脱 (正例)・個表記方針・冬天/夏天・父母。
5. **指摘 25–30 (構造・schema・正例)** — 不整合総括・蛮/老・好多・letter 構造・羅馬字 schema・浪向。

これらを順次適用することで、ユーザーが申し立てた「現状の `wuu` データが実際に話される上海話と乖離している」という根本的問題が解消される。修正範囲は約 45 セル (うち重複なし約 35 セル) で、**`wuu` 100 セル中の 3.5 割が何らかの普通話干渉を含む** という診断。
