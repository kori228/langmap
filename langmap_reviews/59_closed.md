# LangMap データレビュー #59 — 福州話 / 閩東語 (Eastern Min / Foochow, `cdo`) 単一セル — 北京語干渉と閩南語 (Hokkien) 写しの監査

## レビュアー自己紹介 (ペルソナ)

林依妹 (Lìng Ĭ-muói, handle: **@bangucê_imuoi**)。福州市倉山区生まれ・育ちの福州話母語話者。家庭言語は 100% 福州話、両親共に老福州市民で、祖父は閩劇 (Mìng-kiŏk) の俳優。福建師範大学文学院言語学修士、修士論文「現代福州方言詞彙の北京語干渉に関する社会言語学的調査」(2012)。Bàng-uâ-cê (平話字、BUC) は教会経由で祖母から学び、Foochow Wikipedia (cdo.wikipedia.org) の編集経験あり。常用文献は **馮愛珍《福州方言詞典》(江蘇教育出版社 1998、以下「馮典」)**、**陳澤平《福州方言研究》(福建人民出版社 1998)**、**李如龍《福州方言研究》(福建人民出版社 1994)**、Norman *Chinese* (CUP 1988) Ch.9、**李如龍/梁玉璋/鄒光椿/陳澤平《福州方言詞典》(福建人民出版社 1995)**、**福州市地方志編纂委員會《福州市志・方言志》(方志出版社 1998)**、Bàng-uâ-cê 正書法ガイドライン (Maryknoll Fathers 1929; 福州方言拼音方案 1994)、Norman 1988 §9.3「Eastern Min」(Foochow が Min 諸方言から早期に分岐し、Hokkien (閩南) と語彙的に共有はあるが **聲調 7→7、文白二讀の分布、變韻 (tone sandhi-induced vowel alternation) の有無** で截然と異なる旨明記)。

今回のレビュー軸は、ユーザーが直接申し立てた **「現状の `cdo` データが (a) 北京語語彙の音訳借用 (Mandarinism) と (b) 閩南語 (Hokkien) 語彙の写し (Hokkienism) で構成され、本物の福州 口語 (Mìng-ngṳ̄ kō̤-ngṳ̄) と乖離している」** という決定的フィードバックの検証。具体的には以下 5 カテゴリ:

1. **(α) Mandarinism — 北京語語彙が福州字音だけ纏わせて混入**:告訴 / 漫遊 / 改善 / 經歷 / 珍貴 / 廟宇 / 美麗 / 智慧型 / 超市 / 父母 / 故事 / 有趣 / 後壁 (← 閩南) / 多 / 冬天 / 正在 / 賞花 等
2. **(β) Hokkienism — 閩南語 (台語) の語彙・字面が福州話に流入**:**仔 (-á, 閩南の縮小辞)** が **囝 (-giāng, 福州固有の縮小辞)** を駆逐 (鳥仔・裙仔・邊仔・囝仔・慢仔・跤踏仔)、**家己 (ka-tī, 閩南 1sg 再帰)** が **自家 (cê-gă)** を駆逐、**𪜶 (in, 閩南 3pl 専用字)** が **伊各儂 (ĭ-gáuk-nè̤ng)** を駆逐、**媠 (suí, 閩南「美しい」)** が **俊 (cùng, 福州固有)** を駆逐、**後壁 (āu-piah, 閩南)** が **後勢 (âu-sié) / 後背** を駆逐、**邊仔 (pinn-á, 閩南)** が **邊頭 (biĕng-tàu)** を駆逐
3. **(γ) 文白選択 — 文讀字 (literary) が日常口語の白讀字を駆逐**:父母 (文) vs 爹娘 (白)、廟宇 vs 廟、改善 vs 改進、經歷 vs 經驗、珍貴 vs 寶貴、日落 vs 日頭落山、語 vs 話、賞花 vs 看花
4. **(δ) 量詞の北京語化**:部 (電影) → 齣 (puó̤k)、座 (橋) → 條 (dèu)、道 (題/菜) → 條 / 盤
5. **(ε) Bàng-uâ-cê (羅馬字) 注記の全面不在** — 福州話表記は **漢字 + BUC** が教会以来の正書法 (Maryknoll 1929、福州方言拼音方案 1994) で、漢字単独では **8 聲調 (陰平 55 / 陽平 53 / 上聲 33 / 陰去 213 / 陽去 242 / 陰入 24 / 陽入 5) の判別不能・變韻の指定不能**。本案件は grand convention 議題に昇格

確信度: 【確実】= 馮典・陳澤平の記述と母語直感が一致し、現行表記が日常福州話として明確に不自然 (Mandarinism または Hokkienism として明確分類)、【蓋然】= 文白差・register 差で迷い余地あり、【要検討】= schema 拡張 (BUC 併記) 等の grand convention 案件。

## 検証範囲

- **対象**: `data.js` 中の `cdo` セル 全 100 文 (`node -e "...; SENTENCES.forEach(...)"` で全件抽出済、`/tmp/cdo_dump.txt` 保存)

### 26. Bàng-uâ-cê (羅馬字 / BUC / 平話字) 注記の全面不在 — メタ指摘、grand convention 議題 【要検討】

**Current:** 全 100 文の `cdo` セルが **漢字のみ**、Bàng-uâ-cê (BUC、福州方言拼音方案 1994 制定の福州話正式羅馬字) 注記ゼロ。

**Problem:** 福州話の表記体系は **漢字 + BUC の併記** が教会以来の正書法 (Maryknoll Fathers 1929 *Bàng-uâ-cê Romanized Foochow Catechism*、福州方言拼音方案 1994、Klöter & Snow *Eastern Min Phonology and Romanization* §4)。学習者向けデータセットとして:
- 漢字のみでは **7 声調 (陰平 ĭ / 陽平 ì / 上聲 ī / 陰去 í / 陽去 ê / 陰入 ék / 陽入 ĕk) の判別不能**

- **Q3**: 既存の中国語系セル (zh-cmn 等) との拼音注記方針整合

## まとめ — 支配的欠陥と優先対処

**支配的欠陥 (dominant defect): `nan` (台語) セルからの機械コピーによる Hokkienism + 北京語語彙の音訳借用 (Mandarinism) の二重混入 — 「Hokkien-as-Foochow」現象**。具体的には:

## Dev response — round 1 (2026-06-04)

林依妹先生、馮愛珍《福州方言詞典》(1998) と陳澤平 (1998) を主拠とする 26 件 (実エディット 40 件) のご指摘、すべて受諾します。とくにご診断の **「Hokkien-as-Foochow」現象 — `nan` (台語) セルから機械コピーしたうえで福州字音だけ付け替えた痕跡** は、当方の差分検査でも以下の点で決定的に確認されました:

- **閩南語専用字の貼り付け**: `𪜶` (#72)、`媠` (#12/#38/#65) — いずれも教育部臺灣閩南語常用詞辭典立項の Hokkien 固有字で、馮典には立項なし。これらが福州字音 (BUC) では発音不能であるという林先生の指摘は、文字列起源が `nan` セルである強力な証拠になります。
- **縮小辞の系統的混用**: 仔 (-á) が 7 セル (#23/#26/#34/#42/#63/#75/#95) に対し、福州固有の 囝 (-giāng) は #18「袋囝」1 セルのみ — 1:7 の比率は、福州話母語データならありえず、Hokkien 母体のコピー痕跡と整合します。
- **Hokkien 親族・方位語の浸透**: 家己 (#90)、後壁 (#97)、物件 (#43)、老爸 (#25) — これらはいずれも閩南語固有形で、福州話は 自家 / 後勢 / 物甡 / 老爹 が馮典標準。
- **Mandarinism の二重混入**: そのうえに 告訴 / 正在 (正著) / 真多 / 冬天 / 超市 / 智慧型 / 賞花 / 副詞「地」/ 量詞 部・座・道 が文讀風に紛れ込み、福州話の口語基層 (講・著・盡濟・寒天・超級市場・智慧・看花・齣・條・盤) を覆い隠している。
