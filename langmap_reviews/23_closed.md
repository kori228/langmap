# LangMap データレビュー #23 — Arabic — MSA & spoken varieties

## レビュアー自己紹介 (ペルソナ)

ハンドルは **@ʕarabiyyāt-Nour**（本名 Nour Haddad）。レバノン・ベイルート生まれ、米国ジョージタウン大学アラビア言語学博士。専門は現代アラビア語方言地理学（dialectology）と話し言葉のコーパス分析。MSA については Ryding *A Reference Grammar of Modern Standard Arabic* と Wehr 辞典を、方言については Woidich *Das Kairenisch-Arabische*（エジプト）、Holes *Modern Arabic: Structures, Functions, and Varieties* および *Dialect, Culture, and Society in Eastern Arabia*（湾岸）、Cowell *A Reference Grammar of Syrian Arabic*（レバント）、Harrell *A Dictionary of Moroccan Arabic*（モロッコ）、Singer *Grammatik der arabischen Mundart der Medina von Tunis*（チュニジア）を主要典拠とする。実地調査経験はレバント（ar_lev）とエジプト（ar_eg）が最も厚く、この2方言については語彙・統語・音韻のいずれも母語話者レベルで断定できる。湾岸（ar_gulf）・イラク（ar_iq）・モロッコ（ar_ma）・チュニジア（ar_tn）・スーダン（ar_sd）はコーパスと文献および現地協力者の確認に基づいて判断する。本データは全体として方言の弁別語彙（قصاد / يمّ / وايد / بزاف / برشا など）をよく押さえており質が高いが、いくつかの確実な誤りと方言整合性の問題を検出した。

## 検証範囲

ar, ar_eg, ar_gulf, ar_iq, ar_lev, ar_ma, ar_sd, ar_tn の8変種すべてについて、#1〜#100 の全文を精査した。特に (a) 動詞接頭辞（b-未完了 / ʕam 進行 / راح・رح・غادي 未来）の方言整合性、(b) 弁別語彙の正しさ、(c) セグメントの役割文字（role-letter）が当該方言の実際の語順を正しく表しているか、(d) 隣接同一 role-letter の重複や欠落、を重点的に確認した。クエリツール (.wf_langmap_query.mjs) で #1, #3, #5, #20, #50, #76, #84 など疑義のあるセルの現値を逐一照合した。

## 指摘事項 (Issues found)

### 1. #20 ar_gulf — 進行・継続のマーカーが欠落（C セグメント欠落）
- **Current:** A:「من الصبح」 B:「المطر ينزل」
- **Problem:** 原文「It has been raining since this morning（今朝からずっと降っている）」の継続相を表す C 役（لا تزال / لسه / لسّا / مازال に相当）が湾岸のみ完全欠落している。他の全変種は C を持つ（ar_eg「لسه」, ar_lev「لسّا」, ar_iq「صارت」, ar_ma「مازال」 等）。湾岸でも継続を出すなら「لِسّه / لِسَع」または「من الصبح والمطر ينزل」の様な構文が自然で、現状は単なる「朝から雨が降る」になり継続のニュアンスが落ちている。また B「المطر ينزل」内に主語＋動詞が融合しており、他変種の B（المطر＋進行動詞）と粒度は揃うが C を欠く点が問題。
- **Proposed fix:** C:「لِسّه」 B:「المطر ينزل」 A:「من الصبح」（語順は C→B→A が湾岸として自然）。最小限の修正としては C:「لِسّه」を文頭に補う。
- 【蓋然】

### 2. #1 ar_sd — 関係詞 E に動詞が融合し、G セグメントが消失

### 3. #76 ar_tn — 「幸せに感じる」の表現が不自然・冗長（D/E 分割）

### 4. #5 / #18 ar_sd — 「خلاص」を程度副詞（C: very）として使うのは不自然
- **Current（#5）:** A:「القهوة」 B:「دي」 D:「زاكية」 C:「خلاص」 ／ **（#18）** A:「الشنطة」 B:「دي」 D:「غالية」 C:「خلاص」
- **Problem:** 「خلاص」は本来「終わり・もういい・OK」の意で、程度強調「とても（very）」には用いない。スーダン口語で「とても」は「شديد」「خالص（KHaaliS, エジプト借用で /xaːlis/）」または「مرة」が一般的。表記「خلاص /xalaːs/」のままだと意味が変わる。#12, #50, #87 でも ar_sd は同じ「خلاص」を C/強調に充てており系統的。
- **Proposed fix:** C:「خالص」（語末を ص の前を ا にして /xaːliṣ/ = エジプト・スーダン共通の強調詞）または C:「شديد」。少なくとも「خلاص」→「خالص」の綴り修正。
- 【蓋然】

### 5. #84 ar_iq — 「風邪をひいた」の語彙・語順が不自然

- **Proposed fix:** A:「هية」 B|C:「جاها برد」 D:「الأسبوع」 E:「الماضي」（または D|E:「الأسبوع الماضي」）。「ردت رشح」は明白な誤り。
- 【確実】
