# LangMap データレビュー #03 — Koreanic — Standard, regional & historical

## レビュアー自己紹介 (ペルソナ)

**한도윤 (Han Do-yun / @hanmaru-ling)**、韓国語史・方言学を専門とする記述言語学者。ソウル大学校で中世韓国語形態統語論により博士号取得、その後、国立国語院の方言調査プロジェクトと延辺大学校朝鮮語言文学系の客員研究で東北方言・육진(yukchin)방언の臨地調査に携わった。常用する典拠は『표준국어대사전』『우리말샘』、安秉禧・李基文『國語史槪說』、李翊燮『방언학』、소신애ほかの제주어/육진방언記述、北朝鮮側『조선말대사전』(社会科学出版社)、および15世紀文献(『釋譜詳節』『杜詩諺解』『訓民正音諺解』)の原文校訂テキスト。最も自信があるのは **ko_mid(中世韓国語)・ko(標準語)・ko_jeju(済州語)** の三者で、ko_em(近代/開化期書記)とko_yb(延辺)については記述資料が薄い領域につき確実度を抑えめに付与した。本レビューでは、標準語との「綴り直し」に堕している方言行に各変種の*有名な弁別的特徴*が出ているかを最重視した。

## 検証範囲

7言語(ko, ko_bus, ko_em, ko_jeju, ko_kp, ko_mid, ko_yb)× 全100文を通読し、`.wf_langmap_query.mjs` で疑義のあるセル(#3, #5, #26, #35, #42, #62, #75, #80, #95 ほか)をライブ照合した。とくに (1) 方言弁別特徴の有無、(2) 膠着接辞・補助用言のセグメント切りと役割letter整合、(3) 北朝鮮(ko_kp)・延辺(ko_yb)の正書法と語彙、(4) 中世語(ko_mid)の表記・語形の妥当性、を重点的に点検した。全体として標準語(ko)・北朝鮮(ko_kp)・釜山(ko_bus)行は概ね良好。問題は主に ko_jeju の文末法・語尾不一致、ko_mid の一部破格語形、および数文の役割letter整合に集中している。

## 指摘事項 (Issues found)

### 1. #5 ko_jeju — 述語セグメントが脱落し、形容詞「おいしい」が消えている 【確実】
**Current:** A:「이」 B:「커피는」 C:「하영」 D:「마씀」
**Problem:** D に入っている「마씀(/마씸)」は済州語の*終結補助辞(丁寧の念押し)*であって「おいしい」という述語語幹を一切含まない。他の全変種は D に「맛있어요/맛있다/맛있슴다…」と*語幹+語尾*が入る。済州語でも「맛 좋수다 / 맛셔마씀」のように述語が必要。現状は「この コーヒーは とても (ね)」で文が成立せず、しかも #43 で「마씀」を C(おいしい)に流用しているのと矛盾している。
**Proposed fix:** D:「맛 좋수다」(または D:「맛셔마씀」)に差し替え。役割は D のまま。

### 2. #26 / #75 ko_jeju — 「子供たち」に侮蔑語「놈팡이덜」が誤用されている(語義のずれ) 【確実】
**Current (#26):** A:「놈팡이덜이」 E:「공원」 B:「‌디서」 C:「놀곡」 D:「이수다」  /  **(#75):** A:「놈팡이덜은」 B:「헤염치는 거」 C:「배우곡 이수다」
**Problem:** 「놈팡이」は標準語でも済州語でも「遊び人・ろくでなし・ヒモ」を指す侮蔑語で、「子供たち」の意ではない。済州語で「子供」は普通 **「아이/애기」**、複数なら「아이덜/애기덜」。「놈팡이덜이 공원에서 놀고 있다」では「ろくでなし共が公園で遊んでいる」という別文になる。#75 も同断。標準語(ko)・北朝鮮(ko_kp)が「아이들이/아이들은」と正しいだけに、済州語だけ語義が逸脱している。
**Proposed fix:** #26 A:「아이덜이」、#75 A:「아이덜은」 に修正。

### 3. #3 / #8 / #42 / #86 / #92 / #80 ko_jeju — 進行・現在語尾「-엄」が語幹露出のまま終結辞を欠く 【蓋然】
**Current (#3):** … C:「먹엄」  /  **(#42):** … C:「입엄」  /  **(#86):** … C:「쓰엄」  /  **(#92):** … C:「먹엄」  /  **(#8):** … C:「만들엄」  /  **(#80):** … E:「믿엄」
**Problem:** 済州語の進行/未完了の標識「-엄ㅅ-」は終結辞「-어/-저/-수다」等を伴って初めて文が完結する(例: 먹엄ㅅ어, 먹엄저, 먹엄수다)。語幹露出形「먹엄」単独では平叙終結にならず宙吊りになる。同データ内の他の済州語行は「-수다/-주/-우다」を正しく付けているのに、このグループだけ終結辞が脱落しており、変種内で文末法の処理が割れている。
**Proposed fix:** 平叙現在なら C:「먹엄ㅅ어」または C:「먹엄저」、丁寧体に揃えるなら C:「먹엄수다」。#8/#42/#86/#92/#80 も同様に終結辞を補う。

### 4. #80 ko_mid — F/B の分割が破格で「가 호미」という非文断片が生じている 【確実】
**Current:** A:「나ᄂᆞᆫ」 F:「남의 나라」 B:「가 호미」 G:「보ᄇᆡ로온」 C:「녁기미」 D:「이라」 H:「고」 E:「밋ᄂᆞ니라」
**Problem:** 「留学」を「남의 나라(F)+가 호미(B)」と訳し、主格助詞「가」を後続名詞節側に貼って B:「가 호미」としている。「가 호미」は中世語の解析として成立しない(行く=가-の名詞形は「가미/감」、「호미」は한다の名詞形)うえ、助詞「가」が文節頭に立つ非文。さらに C:「녁기미」は「너기-(思う)」の名詞形であって「経験」の訳になっておらず、原文「貴重な経験だと信じる」と意味がずれる。
**Proposed fix:** 留学を一語化: F|B:「ᄂᆞᆷ의 나라 가미」(他国へ行くこと)。経験は中世口語に適語が無ければ漢語のまま C:「經驗이」とし、G:「보ᄇᆡ로온」 C:「經驗이」 D:「라」 H:「고」 E:「밋ᄂᆞ니라」とするのが穏当。少なくとも B:「가 호미」の助詞分離は解消すべき。

### 5. #95 ko_yb — A/B が正体不明の綴り(「재앵교로」「변공실에」)で延辺語として成立しない 【確実】
**Current:** D:「나는」 A:「재앵교로」 B:「변공실에」 C:「감다」
**Problem:** A「재앵교」は「自転車」の延辺語形ではない(延辺でも基本「자전거」)。中国語借用を出すなら「自行车 → 자행거」系はあり得るが「재앵교」は出典不明の誤記。B「변공실」も「办公室(bàngōngshì, 事務室)」の音訳としては声母が誤りで、正しくは **「판공실」**。延辺語は中国語借用語の多用が弁別特徴だが、ここはその特徴が*綴り誤り*として現れてしまっている。
**Proposed fix:** A:「자전거로」(標準同形でよい)、B:「판공실에」(借用を活かす)または B:「사무실에」。C:「감다」は語尾「-ㅁ다」で延辺らしく問題なし。

### 6. #62 ko_yb — 役割 D に完了副詞「다」が混入し、他変種と分割が不一致 【蓋然】
**Current:** A:「나는」 B:「이 책을」 C:「월말까지」 D:「다 읽구」 E:「싶슴다」
**Problem:** D の本動詞は「읽-」で、他の全変種は D:「읽고/읽고져/닑고져」と動詞のみ。ko_yb だけ完了副詞「다(全部)」を D に含めるが、対応する副詞は ja/en にも他韓国語変種にも無い。役割整合上、D は動詞だけにすべき。
**Proposed fix:** D:「읽구」 E:「싶슴다」(「다」を削除)。原意を保つなら「다」を別セグメント(色なし)として分離。

### 7. #42 ko_kp — 「양복치마」が「赤いドレス」の訳として不適切 【蓋然】
**Current:** A:「그이는」 B:「항상」 D:「붉은」 E:「양복치마를」 C:「입는다」
**Problem:** 『조선말대사전』で「양복치마」は「洋服のスカート(ツーピースの下衣)」を指し、ワンピース型の「ドレス」全体ではない。「붉은 양복치마」だと「赤いスカート」となり、原意(1着のドレス)とずれる。北朝鮮でも「원피스」は通用するため、それを用いるのが自然。
**Proposed fix:** E:「원피스를」(D:「붉은」 E:「원피스를」)、または E:「양장을」。

### 8. #100 ko_mid — 「모든 거세」「고맙ᄂᆞ이다」が古綴方針と不整合・時代錯誤の懸念 【蓋然】
**Current:** A:「모든 거세」 B:「고맙ᄂᆞ이다」
**Problem:** 同データの中世語行は #52「져ᄌᆞ에셔」, #66「누ᄂᆡ로」など徹底した古綴・分綴を採るのに、ここだけ「모든」(中世語は「모ᄃᆞᆫ」)、現代的縮約「거세」を用い表記方針が不統一。また「고맙-」は中世語ではむしろ「고마ᄫᆞ-(尊い・かたじけない)」系で、現代語幹「고맙다」をそのまま使うのは時代錯誤。
**Proposed fix:** A:「모ᄃᆞᆫ 거세」、B:「감사ᄒᆞ노이다」または「고마ᄫᆞ이다」。

### 9. #76 ko_jeju — C セグメントに主語「나는」が紛れ込み、役割整合が崩れている 【蓋然】
**Current:** A:「음악을」 B:「들으민」 C:「나는」 D:「행복험수다」
**Problem:** 役割letter C は本文系列では「条件・時の接続(때/저기/기분이)」位置。ko_jeju は B:「들으민(聞くと)」で既に条件接続を表したうえ、C に主語代名詞「나는」を入れており、A(音楽を)も含めた役割対応が崩れる。標準語に倣えば B:「들을」 C:「때」 の構成。
**Proposed fix:** 標準語構成に揃え C:「때」相当を置くか、「나는」を主語 A 側へ再ラベル。少なくとも C に主語を置く現状は不可。

### 10. #29/#64/#86 ko_em — 外来概念の置換語が文間で非一貫(同語別義) 【要検討】
**Current:** #64 B:「내 전음기를」(=スマホ) / #86 B:「전음기를」(=スマホ) / #46 C:「먼데 이르는 기계로」(=電話)
**Problem:** ko_em(開化期書記)の外来概念置換方針自体は妥当だが、「전음기」を #64/#86 でともにスマホに当てる一方 #46 では電話を「먼데 이르는 기계」と別語で表すなど、同概念に別語/同語に別概念が混在し基準が揺れる。
**Proposed fix:** スマホは「손에 드는 전음기」等で電話機と区別、または ko_em の置換ポリシーを文間で統一。明確な誤りではないため低確信。

### 11. #88 ko_em — 「기관의 지혜(機関の知恵)」が AI の訳として苦しい 【要検討】
**Current:** A:「기관의 지혜는」 B:「온 세상을」 C:「고쳐 가니라」
**Problem:** 開化期語彙として「機関(メカニズム)の知恵」は意味が取りにくい。ko_mid は AI を「지혜로ᄫᆞᆫ 얼개」と訳しており、ko_em と語基が揃っていない。
**Proposed fix:** 任意。整合を取るなら ko_em も「슬기로온 얼개」「만든 슬기」等に。創作的等価語の選択であり低確信。

### 12. #51/#73 ko_kp/ko_jeju — 所有格(E)の融合/分離が変種内で非一貫 【要検討】
**Current:** #73 ko_kp A|E:「고양이는」(所有省略) / #50・#65 ko_kp は E:「내」と分離
**Problem:** 所有「私の(E)」を主語に融合する文(#35, #51, #73)と分離する文(#50, #65)が ko_kp 内で混在。文화어として「내 고양이」を出さない選択も可だが、家族内一貫性の観点で要検討。
**Proposed fix:** 所有の表出/非表出ポリシーを統一。誤りとまでは言えず低確信。

### 良好と判断した点
- ko(標準語)・ko_kp(文화어)・ko_bus(釜山)の大半は語彙・語尾・正書法とも妥当。ko_bus の「-ㅂ니데이/-노/억수로/아이가/-ㅁ더」、ko_kp の두음법칙(래일/녀동생/려관/료리·로동)・「뻐스」など弁別特徴が正しく反映されている。
- 膠着接辞の ZWNJ 接頭(H:「‌어버렸다」, F:「‌에서」, D:「‌에」等)は #52/#46/#9 などで一貫して付与され、プロジェクト方針に沿う。
- ko_yb の終結語尾「-ㅁ다/-슴다/-슴까」と接続「-구/-믄」、ko_em の「-니라/-나이다/-도다」、ko_mid の「-노라/-ᄂᆞ니라/-도다」は各レジスターとして概ね正確。
